1、IE进行数据查询时，访问同一个页面时每次会从缓存中取值不会实时获取：========1、如果是ajax请求方式，则设置$.ajaxSetup({cache:false})  2、可以在请求路径中添加一个随机参数，Math.random()(0-1.0的随机数)，这样每次则为不同的请求连接，IE将会进行再次请求

2、当jsp页面变更以后，其他页面可以实时获取变更之后的信息，IE无法获取必须清空缓存才可以获取到=====设置IE的临时性文件读取方式为：每次访问网页时----校验该网页的较新版本（https://jingyan.baidu.com/article/9f7e7ec059aad46f281554db.html）
