@slf4j注解的使用
	https://www.cnblogs.com/weiapro/p/7633645.html
	若不想每次都声明一个Logger对象，则可以在测试类加入注解@slf4j 然后 无需声明对象可直接使用内置对象log即可，同时支持手动创建其他log对象
	
日志框架
==========================================================
日志接口：jcl（java comm logging）  slf4j 
日志实现类：log4j log4j2 logback jul(java util log) 

日志框架：通过日志接口和实现类的配合 形成日志框架
